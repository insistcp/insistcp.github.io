Tensorflow学习笔记-第一讲Tensorflow实现softmax Regression识别手写数字

发表于 2017-05-31 | | 阅读次数

一.问题描述

MNIST手写数字识别作为机器学习领域的Hello World.了解它的实现原理十分有必要，今天我们就来学习使用tensorflow实现手写数字识别
MNIST数据集：是一个机器视觉数据集，由几万张28像素*28像素的手写数字组成，图片只包含灰度值，我们的任务就是对这些手写数字进行分类，转成0-9一共10类。

二.实现原理

1：数据处理
由上面描述我们知道每一个样本具有2828维的特征，我们将这个矩阵拉成一个1维向量，使得其成为784列的向量。则训练集的特征就是一个55000784的tensor。训练数据集的label则是一个55000*10的tensor。
2：算法设计
我们使用SoftMax Regression的算法对手写数字构建分类模型，我们最终的分类是一个0-9一共10类，Softmax Regression会对每一个类别估算一个概率，最后选出概率最大的那个数字作为模型的输出结果。

$$\sum_{i=1}^n w_ij*x_j+b_i$$

将所有特征提取出来，然后再对所有特征进行softmax(x) = normalize(exp(x))进行处理。
$$softmax(xi)={exp(xi)\over \sum_{v=1}exp(xi)}$$
